Network Statistics SDN Application runs on-top of a OpenDaylight SDN Controller.
It tracks port, table and flow statistics of all the nodes added to OpenDaylight. Run this application on tomcat locally and supply the credentials below to see it in action.

This is a pure jQuery/Bootstrap and HTML based application without any backend in itself.







Track Network Statistics

Network Devices

This section shows all the device loaded into OpenDaylight. I have created a thumbnail for these devices.

You can very well use any JS plotting frameworks to show the network topology also



Please enter OpenDaylight credentials first.

Port Statistics

This section shows the port statistics of the selected device.

You can intelligently show which ports are running maximum traffic and are at risk etc.
There are numerous use cases that open up with this data



Please select a device to view port statistics.

Table Statistics

This section shows the OpenFlow table statistics of the selected device.

This information is helpul in pre-configuring the switch and knowing the switch capacity.



Please select a device to view table statistics.
